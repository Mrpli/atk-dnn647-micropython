## 1.2 测试

在按照1.1接线完成后，请详细检查电路，确认没有问题请打开打开串口调试助，配置模式为波特率：115200、数据位： 8 校验位：none 停止位：one 流控：none

配置信息如下：

确认没有问题请按一下EN按键RST复位键（300m左右的低电平），打印信息如下：

显示 ready 证明启动正常。

常见问题：

1、按键没有反映

（1）请检查模组供电电源是不是3.3V/800ma的电源；

（2）请检查一下TX和RX是不是接错位置；

（3）检查一下端口号选择是否正确，有没有关闭串口调试助手的端口。

2、按键全是乱码

（1）请和模组厂家确定模块的波特率是多少;

（2）请检查和模组通信的USB转TTL是不是兼容性问题，建议使用的是CH340和CP2102芯片做的USB转TTL.

（3）请确定模组里面的程序是不是有问题。（后面会讲到如何对模组进行烧录固件）

## 2.1 模块 AP 模式下做 TCP serve

AT+CWMODE=2 开启 AP 模式（串口助手）

AT+CWSAP="BK7238","0123456789",11,0 设置模块的 wifi 和密码（串口助手）

AT+CIPMUX=1 打开多连接（串口助手）

AT+CIPSERVER=1,8080 设置模块服务器端口（串口助手），

打开手机开始进行设置： 请打开电脑去连接模块建立的AP热点BK7238。并且在打开网络调试助手，输入连接模块的 ip 和设置的端口。如下图模块默认的 ip 为 192.168.4.1（可以通过电脑WiFi的属性查看），端口为自己设定的 8899 （网络调试助手）

AT+CIPSEND=0,20 进入数据发送模式为20个字节（串口助手）

\> 进入发送模式（串口助手）

www.sparkleiot.com 发送的 20 个数据（串口助手）

# 三、常用指令篇

## 3.1 AT 常用

1、读取 IO 状态

AT+CIOREAD=26 15 为 GPIO15，返回值 0:LOW（低电平） OK

2、设置 IO 口状态

AT+CIOWRITE=26,1 //本指令是星火内部指令 2 为 GPIO 口号，1 为高电平，0 为低电平。

3、智能配网（手机app 配网）

AT+BLUFI=1 ，要用乐鑫提供 APP 。上乐鑫官网下载 ESP-BLUFI ： [](http://espressif.com/zh-hans/support/download/documents)[App｜乐鑫科技 (espressif.com.cn)](https://www.espressif.com.cn/zh-hans/support/download/apps?keys=&field_technology_tid%5b%5d=719)

4、查询芯片 ID

AT+CSYSID

+CSYSID:CHIP:1V2OAXDZ65;FLASH:AT&BK7238;KEY:china

5、设置波特率

AT+UART_DEF=9600,8,1,0,0 修改串口波特率并保存到 flash，第二次上电有效模块不支持硬件流控。

6、保存 TCP/UDP透传指令

AT+SAVETRANSLINK=1, ”192.168.31.20”,1002, “TCP” 1 开机进入透传模式， 192.168.31.20 代表远端 ip，1002 远端端口。

7、 TCP服务器

AT+CIPSERVER=0 返回 OK

8、恢复出厂设置（配网失败可以用这个指令）

AT+RESTORE

9、打印 AP 的 mac

AT+CIFSR

10、设置 STA 模式 MAC 地址

AT+CIPSTAMAC="18:fe:35:98:d3:7b"

# 四、模块固件烧录篇

程序下载是在原来接线的基础上把 TXD 拉低进行烧录（芯片内部已经处理），建议使用最新版本的下载软件。电源要求 3.3V/800MA 的稳定供电，选择如下图。

1.选择正确的COM口和波特率；

2.文件不要选错(AT_115200_V3.0.7)；

3.烧录速度建议用 115200，烧录速度越快对串口芯片要求越高；

4.接线一定要尽量短和模组尽可能的独立供电。

注意:

选择好之后，就可以开始烧录了，点击 烧录，在给模块复位。注意不要给串口断电。如果还有没有进入下载中，就给复位脚几次低电平脉冲看有没有效果。在进行下载时，只要接线没有问题，谨记接线也不要太长基本上是没有问题。一定要注意 TX 和 RX的通信处的电源。烧录不成功 90%以上都是电源问题。
